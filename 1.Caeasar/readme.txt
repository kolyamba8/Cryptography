Работа выполнена в среде Visual Studio 2008 на языке c#.

Файлы:
Alphabet.cs - содержит описание класса Alphabet
Caesar.cs - содержит описание класса Caesar
Program.cs - содержит описание класса Program и функию main()
Form1.Designer.cs - содержит разметку окна
Form1.cs - содержит обработчики событий окна
1.Caesar.csproj - аналог make файла для утилиты MSBuild, содержит данные, необходимые для компиляции програмыы

Компиляция производится с помощью утилиты MSBuild
